# Leafpad

A clean, ad-free markdown editor with live preview.

**Live:** https://leafpad.vercel.app

## Features

- Live split-pane preview
- Keyboard shortcuts (⌘B, ⌘I, ⌘K, and more)
- Export to .md, .html, or copy HTML
- Live stats (words, characters, lines, reading time)
- Multiple themes, editor themes, and fonts
- GFM support (tables, task lists, strikethrough)
- Syntax highlighted code blocks
- Auto-save to localStorage

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

## Getting Started

```bash
npm install
npm run dev
```
